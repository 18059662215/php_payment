一个PHP语言写的微信支付和支付宝支付插件

# 环境要求

最低php5.4,建议在`PHP7`上运行以获取最佳性能。

#SDK`相关介绍#

目前有两大支付类，未来会增加更多，当然不至有支付，还有更多小工具！

* **微信支付**

微信扫码支付，jsapi支付，订单查询，订单退款

* **支付宝支付（当面付）**

扫码支付，订单查询，订单退款
